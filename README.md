# Sayla

Sayla is an AI communication assistant that helps people write and reply with the right tone, context, and language.

It is designed for moments when you know what you want to say, but worry that it may sound too direct, cold, awkward, or unnatural — especially at work or in a foreign language.

## Features

- Rewrite your own message
- Generate thoughtful replies
- Choose context, tone, and target language
- Supports English, German, and Korean
- Tone insight for communication risk and improvement
- Copy for Slack
- Open as email draft
- Create follow-up reminder
- Guided by a custom Tone Playbook MCP

## Built with

- v0
- Vercel
- Next.js
- Groq
- Tailwind CSS
